# Granting a Service Package to an Organization

## Description
This section illustrates how to grant a service package to an organization.

## Who can perform this function?
Security Administrator

##Steps
1. Log into CIS.
2. Select an organization from your organization hierarchy (My Organization->View my Organization Hierarchy), or select an organization using the search option (Search->Search for Divisions in my Organizationâ€™s hierarchy).   
![](gspo-2.png)
3. Click **add service package**.
![](gspo-3.png)
4. Select a package by clicking on the **Add** button.
![](gspo-4.png)
5. Review the package details. Click **more info** to view services associated with the package.
